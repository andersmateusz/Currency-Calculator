This is The Currency Calculator web app.
This app uses exchange rata API. 
If you want to use this app get your free API key from https://exchangeratesapi.io after registering. 
To use your API key insert it to:
/src/Controller/IndexController.php line 20
as argument for CurrencyManager(_YOUR_API_KEY_HERE).
By default there is already API key, but it might be not up to date when you are reading this.
