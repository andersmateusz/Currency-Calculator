This is The Currency Calculator web app.
This app uses exchange rata API. 
If you want to use this app get your free API key from https://exchangeratesapi.io after registering. 

Warning:
- If you choose a past date for which there is no data at https://exchangeratesapi.io the app will crash. There is no exception for that. There is only a constraint for inserting a future date.
- If the service https://exchangeratesapi.io stops after inserting your API key the app will also crash. No exception.
- If your limit of requests from https://exchangeratesapi.io ends after inserting API key the app will crash. No exception.
